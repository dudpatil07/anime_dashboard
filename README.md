# AnimeVerse 🎌

A stylish, frontend-only anime recommender web application with genre filtering, personal library management, and direct streaming links.

## ✨ Features

- **Discover** — Browse 48 curated anime across 13 genres
- **Favorites** — Heart your most loved anime
- **My Library** — Save anime you plan to watch
- **Watched** — Track completed series
- **Genre Filtering** — Action, Adventure, Comedy, Drama, Fantasy, Horror, Romance, Sci-Fi, Slice of Life, Sports, Supernatural, Thriller
- **Sorting** — By popularity, rating, newest, or title
- **Real-time Search** — Find anime by title or genre
- **Daily Anime Pick** — Spin the wheel to get a unique anime recommendation based on the current date, inspired by randomanime.org
- **User Profile** — Personal stats dashboard with achievements, rating distribution, genre breakdown, activity timeline, and top rated list (inspired by malune.net)
- **Streaming Links** — Direct links to Crunchyroll and Anikai.to for every anime
- **Persistent Storage** — All data saved via localStorage
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Retro Anime Aesthetic** — Neon glow effects, floating particles, glass-morphism UI

## 🛠️ Technologies

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, animations, grid, flexbox, backdrop-filter
- **Vanilla JavaScript** — No frameworks, pure ES6+
- **LocalStorage API** — Data persistence
- **Font Awesome** — Icons
- **Google Fonts** — Orbitron & Rajdhani typography

## 📦 Data

The app includes 48 hand-picked anime with:
- Real MyAnimeList cover images
- Accurate ratings, years, and episode counts
- Genre classifications
- Detailed descriptions
- Direct Crunchyroll & Anikai.to streaming URLs
